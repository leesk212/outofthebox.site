# Format
```
<img src="directory/200x200.png" alt="My Image">
```
* src : 이미지의 경로
* alt : 이미지를 표시할 수 없을 때 출력할 내용
* width : 이미지의 가로 크기
* height : 이미지의 세로 크기
> width와 hieght 하나만 정한다면 비율 맞춰서 따라옴
* loading : 이미지 로딩 방식 (= "lazy": 느리게 로드)(= default: eager)
